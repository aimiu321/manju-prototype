环节3 - 场景图 操作说明
1页面结构
一级页面 场景图列表页，横向卡片布局，左侧缩略图+右侧信息
二级页面 点击卡片进入，场景图预览+场景描述（可编辑）
场景库 顶部入口，半屏弹出，3列网格展示本剧已生成场景
2一级页面操作
点击卡片区域 → 进入二级页面编辑场景描述
点击 生成场景图 按钮 → 校验登录+积分 → 生成该场景图（10积分）
点击 重新生成 按钮 → 积分充足直接执行（无确认弹窗）
点击已生成的场景图 → 全屏放大预览（黑色遮罩+居中图片）
点击 场景库 → 半屏弹出，可引入已有场景图到当前集
点击 返回角色 → 返回环节2，有场景图时弹窗提示
3二级页面操作
主视角角色信息不展示，角色图由系统自动匹配作为五官参考
场景描述 点击进入编辑态，紫色边框高亮，失焦/回车保存，最大200字
编辑已生成场景的描述后 → 自动标记 需重新生成，系统自动重新拼接完整提示词
点击已生成的场景图 → 全屏放大预览
点击 ←返回 → 回到一级页面
4场景图状态
待生成 占位图 + "生成场景图"按钮
生成中 骨架屏动画 + 按钮隐藏
已生成 场景图 + "重新生成"按钮 + 可放大预览
已选择 场景图（来自场景库引入/自动填充）+ "重新生成"按钮
生成失败 ❌图标 + "重新生成"按钮 + 退还积分
需重新生成 场景图仍展示 + 橙色标签+角标 + "重新生成"按钮
5底部按钮逻辑
返回角色 始终显示，有场景图时弹窗提示"角色修改后请重新生成"
一键生成 有未完成/需重新生成的场景时显示，依次生成
下一步 全部场景已生成且无"需重新生成"时，按钮文案变为"下一步"，跳转环节4
6场景库规则
场景库按剧（series）隔离，不同剧集互不可见
场景图生成后自动存入场景库
非第一集：自动从场景库匹配填充（状态"已选择"）
手动引入：点击"场景库" → 选择引入 → 带入场景图+描述
同描述场景不可重复引入
7前序环节联动
环节2角色形象变更 → 返回环节3时，所有已生成场景标记 需重新生成
原因：场景图依赖角色图作五官参考，角色图变更后五官已不准确
调试器中"模拟角色变更"按钮可触发此联动
8提示词自动拼接（用户不可见）
完整提示词 = 画风前缀 + 场景描述 + 角色外貌 + 动作 + 参考图中人物的面部五官和形象
用户只能编辑"场景描述"，其余由系统自动组装
编辑场景描述后，系统自动重新拼接完整提示词
调试器底部可查看自动拼接结果
9积分与登录
场景图生成：10积分/张，生成/重新生成/一键生成均需消耗
生成前校验：未登录→登录弹窗 → 积分不足→积分不足半屏
积分不足半屏：非会员→"开通会员"，会员→"购买积分包"
生成成功扣减积分，失败/超时解冻退还
重新生成无确认弹窗，积分充足直接执行
