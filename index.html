环节1 · 原型操作说明


  

    
一、页面概览

    

      本原型模拟「环节1-内容创作」的完整交互流程，包括：

      ① 表单填写（剧集信息+大纲+剧本）

      ② AI功能（AI优化/AI灵感/剧本拆解）

      ③ 登录/积分/会员体系

      ④ 拆解结果展示 & 续写下一集

      所有交互均为前端模拟，AI返回的是预设demo数据。
    

  


  

    
二、调试面板使用

    

      右侧「调试面板」可切换各项状态，验证不同场景的交互表现：


      

        
调试项操作验证场景

        
登录点击「切换」未登录→触发AI时弹出登录弹窗；登录后正常操作

        
会员点击「切换」积分不足时：非会员→「开通会员」按钮；会员→「去充值」按钮

        
积分点击 0/2/1000→积分不足弹窗；2→刚好够用；100→充足

        
集数点击「重置第1集」切回第1集，大纲恢复可编辑

        
AI自动显示AI生成中显示「生成中」，完成后恢复「空闲」

        
项目「模拟进入」/「清除」模拟从项目进入恢复内容 / 清除保存数据

        
全部重置点击红色按钮所有状态恢复初始值

      

    

  


  

    
三、核心交互流程


    

      
流程A：自行输入 + 提交拆解

      

        
1. 确认右侧调试面板「已登录」「积分≥2」

        
2. 填写漫剧名称（必填）+ 漫剧剧本（必须非空，「开始创作」才会亮起）

        
3. 点击「开始创作(-2)」→ 全屏Loading「AI正在拆解剧本…」

        
4. 展示拆解结果：角色列表 + 分镜列表

        
5. 点击角色/分镜中的「角色描述」「场景图提示词」「视频提示词」可折叠展开

        
6. 非最后一集→「下一集」；最后一集→「下一步」跳转环节2

      

    


    

      
流程B：AI优化

      

        
1. 在剧本/大纲区域输入内容

        
2. 点击「AI优化(-2)」→ 骨架屏Loading

        
3. Loading结束后内容流式输出，替换为AI优化版本

      

    


    

      
流程C：AI灵感

      

        
1. 点击「AI灵感(-2)」→ 无弹窗，直接开始生成

        
2. 骨架屏Loading→流式输出AI生成内容

      

    


    

      
流程D：未登录 → 登录

      

        
1. 右侧调试面板点击「切换」设为「未登录」

        
2. 点击任何AI按钮 → 弹出登录弹窗 + toast「您还未登录」

        
3. 输入手机号+验证码（任意11位+4位以上）→ 登录成功

        
4. 回到页面，用户需自行重新点击按钮（不自动继续）

      

    


    

      
流程E：积分不足 → 会员/充值

      

        
1. 右侧调试面板设「已登录」+ 积分0

        
2. 点击AI按钮 → 弹出「积分不足」半弹窗

        
3a. 非会员 → 按钮「开通会员」→ 跳转会员订阅Tab

        
3b. 会员 → 按钮「去充值」→ 跳转积分充值Tab

        
4. 选择套餐 → 支付 → 积分到账 → 回到创作页

      

    


    

      
流程F：续写下一集

      

        
1. 完成第1集拆解 → 点击「下一集」

        
2. 自动进入第2集：大纲只读（🔒标记），剧本清空

        
3. 导航栏标题变为「内容创作 · 第2集」

        
4. 大纲区域隐藏AI模式切换按钮

      

    


    

      
流程G：项目保存

      

        
1. 点击「开始创作」提交时，内容保存至项目

        
2. 提交后表单锁定只读，内容保留不清空

        
3. 从首页进入 → 空白表单（新建创作）

        
4. 从项目进入 → 自动加载已保存内容（继续编辑）

      

    


    

      
流程H：返回确认

      

        
1. AI生成中点击左上角← → 弹窗「AI正在生成中，确定离开吗？」

        
2. 有内容时点击← → 弹窗「离开创作，记录将丢失，确定离开吗？」

        
3. 无内容时点击← → 直接返回

      

    

  


  

    
四、页面元素与状态说明

    

      
元素状态表现

      
「开始创作」按钮剧本为空灰色置灰，不可点击

      
「开始创作」按钮剧本有内容紫色渐变，可点击

      
提交后底部已提交绿色「✅ 第N集已提交拆解」，表单只读

      
剧本输入框字数>4000红色边框 + 红色字数统计

      
大纲输入框字数>500红色边框 + 红色字数统计

      
大纲输入框第2集+半透明+禁用，显示🔒只读标记

      
AI生成中Loading态骨架屏动画→流式输出文字

      
导航栏标题第2集+变为「内容创作 · 第N集」

      
积分不足弹窗非会员按钮「开通会员」→跳转订阅Tab

      
积分不足弹窗会员按钮「去充值」→跳转充值Tab
