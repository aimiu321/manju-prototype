全局模块 - 我的漫剧 操作说明
1模块概述
"我的漫剧"是用户管理已保存漫剧剧集的全局模块，包含 剧本列表 和 剧集列表 两个页面
入口A 首页"我的漫剧"按钮 → 剧本列表 → 点击剧本 → 剧集列表
入口B 环节5保存成功 → "查看我的漫剧" → 直接跳剧集列表（跳过剧本列表）
2剧本列表页
导航栏：← 返回 | "我的漫剧"
剧本卡片：封面(最新剧集缩略图) + 剧本名 + 类型·集数 + 箭头
点击剧本卡片 → 调用 GET /scripts/{id}/episodes → 进入剧集列表
空状态："还没有漫剧作品，去创作吧" + "开始创作"按钮
未登录点击"我的漫剧"→ 弹出登录弹窗，登录后继续
3剧集列表页
导航栏：← 返回 | 剧本名称（如"仙途缘起"，非"我的漫剧"）
剧集卡片：缩略图(▶) + 剧集名 + 日期·时长 + 编辑按钮 + 下载按钮
点击卡片 → 环节5 只读播放模式（仅"下载视频"按钮）
点击 编辑 → 环节5 编辑模式（完整按钮，可回退环节4~1）
点击 下载 → 校验相册权限 → 保存到相册
点击 + 创作下一集 → 跳转环节1，自动填入剧本大纲（详见第6节）
4只读模式 vs 编辑模式
只读模式（点击卡片进入）：底部仅"下载视频"按钮，← 返回→剧集列表
编辑模式（点击"编辑"进入）：底部完整按钮（重新合成/下载/保存），可←回退环节4→3→2→1
编辑模式下回退：环节5→4(分镜)→3(场景图)→2(角色)→1(内容)，环节1←返回→剧集列表
目的：让用户回看各环节的生成记录（角色形象、场景图、分镜等）
5返回逻辑
剧集列表 ← 返回：入口A→剧本列表；入口B→环节5
环节5 ← 返回：只读模式→剧集列表；编辑模式→环节4
环节1 ← 返回："创作下一集"进入→剧集列表；"编辑"回退→环节2；全新创作→首页
6创作下一集 — 大纲自动带入
自动填入 剧本大纲（outline）→ "剧本大纲"textarea，用户可编辑
自动填入 剧集名称 → 填入当前剧本名称
不带入 剧情设定、画面风格、视频比例 — 由用户重新选择
新集自动关联同一scriptId，共享角色库和场景库
环节1显示黄色提示："已自动填入剧本大纲，可编辑修改"
7登录与积分
查看"我的漫剧"：需登录，未登录弹出登录弹窗
下载剧集：无需登录
创作下一集：需登录（进入环节1时校验）
本模块0积分消耗，进入环节1后按环节1规则执行
8MVP限制
不支持删除剧集（后续版本增加左滑删除+二次确认）
不支持编辑剧集名称（仅保存成功半屏中可编辑）
不支持剧集排序（固定按episodeNumber升序）
