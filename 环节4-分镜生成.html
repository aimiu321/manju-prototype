快捷操作
环节4 - 分镜生成 操作说明
1页面结构
一级页面 分镜生成列表页，纵向卡片布局（左侧视频区+右侧信息行+下方全宽描述）
无二级页面 分镜描述通过底部Sheet编辑，无需跳转二级页
底部操作栏 「生成全部」幽灵按钮 + 「合成视频」渐变紫色按钮（≥2条已生成时可用）
2一级页面操作
点击分镜描述文本 → 底部弹出编辑Sheet
点击 生成分镜 按钮 → 校验登录+积分 → 生成视频（50积分）
点击 重新生成 按钮 → 积分充足直接执行（无确认弹窗）
点击播放按钮 → iOS原生AVPlayer内联播放（卡片区域内播放，非全屏）
点击场景图缩略图 → 全屏放大预览
点击 合成视频 → 弹出分镜选择半屏 → 选择≥2条分镜 → 点击「跳转环节5」→ 携带选中分镜ID跳转环节5
点击 ←返回 → 按视频状态区分弹窗（见第7节）
3分镜描述编辑（底部Sheet）
点击描述区域 → 底部弹出编辑Sheet（文本框+字数统计+取消/确认）
字数限制 500字，超限红色警告，为空不允许提交
编辑已生成视频的描述后 → 自动标记 描述已修改，需重新生成
双一致性约束被删除 → 保存时自动补回 + toast提示
生成中的分镜描述禁止编辑（灰色不可点击）
4视频状态
待生成 「待生成」文字覆盖 + 「生成分镜」按钮（紫色）
生成中 shimmer动画 + 按钮置灰 + 描述禁止编辑
已生成 ▶播放按钮+时长+绿色角标 + 「重新生成」按钮 + iOS原生AVPlayer内联播放
生成失败 ❌图标 + 「重新生成」按钮 + 退还积分
描述已修改 ✎图标+黄色覆盖 + 「重新生成」按钮（紫色）
场景图已变更 ⚠图标+橙色覆盖 + 「重新生成」按钮（紫色）
画风已变更 ⚠图标+粉色覆盖 + 「重新生成」按钮（紫色）— 优先级最高
状态优先级：画风已变更 > 场景图已变更 > 描述已修改
5底部按钮逻辑
生成全部 有未完成/需重新生成的分镜时显示，逐条校验积分依次生成
合成视频 ≥2条已生成时可用，点击弹出分镜选择半屏，已生成视频默认勾选，选择≥2条后点击「跳转环节5」
生成全部时积分不足 → 中断 + toast「积分不足，还差X积分」，已生成的保留
6视频库规则
环节4不设视频库，分镜视频不入库
视频不自动填充，每集分镜视频均独立生成
不支持删除分镜视频（删除会影响合成流程）
有已生成视频时，顶部显示提示条「分镜作品仅在云端保留7天，请及时保存」
7前序环节联动（4种变更类型）
环节3场景图变更 → videoPrompt重新组装 + 视频标记 场景图已变更，需重新生成
环节1画风变更 → 所有videoPrompt重新组装 + 场景图+视频标记 画风已变更，需重新生成
环节2角色外貌变更 → 含该角色videoPrompt重新组装 + 场景图+视频标记需重新生成
环节1剧本内容变更 → 对应videoPrompt重新组装 + 场景图+视频标记需重新生成
重新组装会覆盖用户手动编辑，toast提示「分镜描述已更新」
调试器中「模拟×变更」按钮可触发各联动
8videoPrompt模板化组装
videoPrompt = 画风特征 + 镜头运动 + 画面中主视角位置+动作 + 非参考角色外貌+位置+动作 + 音色公式 + 「对白」+ 氛围音效 + 双一致性约束
主视角角色不写外貌（五官由首帧场景图锚定），非参考角色才写角色外貌
前序变更时按模板重新组装，各部分从结构化字段取最新值
调试器底部可查看组装结果
9积分与登录
分镜视频生成：50积分/条，生成/重新生成/生成全部均需消耗
生成前校验：未登录→登录弹窗 → 积分不足→积分不足半屏
积分不足半屏：非会员→「去充值」跳转会员订阅，会员→「去充值」跳转积分充值
生成成功扣减积分，失败/超时退还
重新生成无确认弹窗，积分充足直接执行
10返回确认（按视频状态区分）
视频生成中 → 弹窗「视频正在生成中，离开将中断进度，确定返回场景图环节吗？」→ 确定后中断+退还积分
有已生成视频 → 弹窗「已生成的视频将保留，确定返回场景图环节吗？」
无任何视频 → 直接返回环节3
11调试面板
登录/会员/积分 → 下拉选择/输入，模拟不同用户状态
快捷场景 → 一键切换新用户/免费用户/会员/积分紧张
全部瞬间生成 → 所有分镜跳过等待直接标记为已生成
分镜1失败 → 模拟网络异常，分镜1标记为失败+退还积分
模拟场景图/画风/角色外貌/剧本内容变更 → 触发对应联动
全部重置 → 红色按钮，所有状态恢复初始值
