环节2 · 原型操作说明
一、页面概览
本原型模拟「环节2-创建角色形象」的完整交互流程，包括：
① 角色列表展示与编辑（一级页面卡片+二级页面详情）
② 角色图生成（文生图/文+参考图 两种模式）
③ 登录/积分/会员体系
④ 角色库（本剧独立）与自动填充
⑤ 参考图替换/删除、放大预览、画风变更联动
所有交互均为前端模拟，AI返回的是预设demo数据。
二、调试面板使用
中间「调试面板」可切换各项状态，验证不同场景的交互表现：

调试项操作验证场景
登录点击「未登录/已登录」切换未登录→触发生成时弹出登录弹窗；登录后正常操作
积分输入数字后回车0→积分不足弹窗；10→刚好够1张；100→充足
会员点击「非会员/会员」切换积分不足时：非会员→「开通会员」按钮；会员→「购买积分包」按钮
角色状态点击预设按钮快速切换全部待生成/已生成/失败/生成中/混合状态
集数模拟点击「第1集/第2集」第1集=3个AI拆解角色；第2集=自动填充角色库角色+1个新角色
画风变更点击「模拟画风变更」所有已生成角色标记为待重新生成+toast提示
相册权限点击「未请求/已授权/已拒绝」切换未请求→首次上传弹系统权限请求；已授权→直接选照片；已拒绝→弹引导去设置
重置点击「重置」所有状态恢复初始值
三、核心交互流程
流程A：生成单个角色形象
1. 确认调试面板「已登录」「积分≥10」
2. 一级页面点击「生成形象」按钮 → 积分充足直接执行（无确认弹窗）
3. 骨架屏动画 → 2秒后展示角色图
4. 状态标签变为「已生成」，按钮变为「重新生成」
5. 也可进入二级页面点击「生成角色形象」按钮，流程相同
流程B：一键生成全部角色
1. 底部点击「一键生成」→ 积分充足直接执行（无确认弹窗）
2. 所有角色并行后台生成（1.5~3秒随机完成）
3. 每个角色独立完成后toast通知「XX形象已生成」
4. 全部完成后底部按钮文案变为「下一步」
流程C：重新生成（无确认弹窗）
1. 已生成的角色点击「重新生成」→ 无确认弹窗，积分充足直接执行
2. 积分不足 → 按会员状态弹出积分不足半屏（非会员→开通会员，会员→购买积分包）
流程D：编辑角色描述/外貌
1. 点击角色卡片进入二级页面
2. 点击「角色描述」或「角色外貌」文本 → 进入编辑态（紫色边框高亮）
3. 修改内容后失焦或回车保存
4. 角色描述最大500字，角色外貌最大80字，超限红色提示，为空恢复原值
5. 编辑已生成角色的描述/外貌后 → 自动标记为待重新生成
流程E：切换生成模式 + 参考图上传
1. 二级页面点击「文+参考图」标签 → 切换为模式2
2. 角色描述自动插入「参考图中人物的面部五官和形象」
3. 显示参考图上传区 → 点击+号 → 弹出选择「拍照/从相册选择」
4a. 首次选择相册 → 弹出系统权限请求「允许访问照片？」→ 允许后打开相册
4b. 权限已拒绝 → 弹出引导弹窗「无法访问相册」→ 点击「去设置」跳转系统设置
5. 选择照片后 → 预览+上传成功，缩略图显示
6. 点击已上传参考图缩略图 → 弹出选择「拍照/从相册选择/删除」
7. 删除参考图 → 自动切换回模式1，角色描述移除参考图文字
流程F：未登录 → 登录
1. 调试面板点击「未登录」
2. 点击任何生成按钮 → 弹出登录弹窗（手机号+验证码）
3. 输入任意11位手机号+4位以上验证码 → 登录成功
4. 自动继续执行刚才触发的生成操作
流程G：积分不足 → 会员/充值
1. 调试面板设「已登录」+ 积分设为0
2. 点击生成按钮 → 弹出「积分不足」半屏弹窗
3a. 非会员 → 按钮「开通会员」→ 跳转会员开通页
3b. 会员 → 按钮「购买积分包」→ 跳转积分包充值页
流程H：角色库引入
1. 顶部导航栏点击「角色库」→ 底部弹出半屏
2. 3列网格展示本剧已生成角色，每个角色右下角「引入」按钮
3. 点击「引入」→ 带入角色图+描述+外貌到当前集，标记「已生成」
4. 同名角色不可重复引入，标签变为「已选择」
流程I：非首集自动填充
1. 调试面板点击「第2集」
2. 自动从角色库提取已有角色填充到当前集（角色图+描述+外貌，标记「已生成」）
3. AI拆解产生的新角色标记为「待生成」
4. 同名角色自动去重（库中数据优先）
流程J：放大预览 & 画风变更
1. 点击已生成的角色图（一级/二级页面均可）→ 全屏预览，点击空白关闭
2. 调试面板点击「模拟画风变更」→ 所有已生成角色标记为待重新生成
3. toast提示「画风已变更，请重新生成角色形象」
流程K：重新分析
1. 底部点击「重新分析」→ 确认弹窗「重新拆解剧本将消耗2积分，继续？」
2. 确认后消耗2积分，重新拆解剧本，角色列表会被更新
流程L：返回环节1
1. 左上角点击←返回 → 确认弹窗「返回后剧本将重新拆解，确认返回吗？」
2. 确认后返回环节1内容创作页面
